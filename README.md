# Coding Project 05: Helpdesk Ticket Board

## Setup steps
In terminal: 
1. cd Documents
2. npx create-next-app@latest helpdesk-board (no TypeScript, Yes Eslint, Yes TailWind CSS, use src/, Yes App Router, No customize alias)
3. cd helpdesk-board
4. npm install
5. npm run dev -> choose http://localhost:3000
